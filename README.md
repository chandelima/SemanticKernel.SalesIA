Este repositório contém uma aplicação .NET Web API que utiliza o Semantic Kernel para incorporar IA generativa e oferecer respostas contextuais a partir de interações com dados da aplicação.

# Tecnologias
.NET 8
Semantic Kernel (Microsoft)
API da OpenAI

# Funcionalidades
Integração com IA Generativa a partir do us do Semantic Kernel para processar e responder a perguntas de forma contextual.
Extensibilidade com Plugins: Expanda as funcionalidades da IA com plugins personalizados que conectam a IA a APIs e dados da aplicação.
